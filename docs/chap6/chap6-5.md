## Les conditions

Comme en JavaScript. On retrouve les tradionnels "if..else"  et "switch...case" :

```js
if(une_conditon) {  
   // La condition est vraie
} else {
   // Elle est fausse  
}
```

Et un switch-case :

```
switch(meteo) {
   case 'soleil': {
      //Il fait beau
      break;
   }
   case 'pluie': {
      // Il fait moins beau
      break;
   }
   default: {
      // devine
      break;
   }
}
```
